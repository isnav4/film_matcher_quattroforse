FILM MATCHER !!!

A H O L A Sono un chatbot che permettere di trovare film basandosi su una o più parole! La mia parte backend è in Python, con Flask a fare da mediatore e un modello Transformer (Sentence-BERT) per analizzare le trame dei film che conosco!

come puoi installarmi ??

copia la repository o scarica la cartella zip. installa le librerie necessarie per farmi funzionare: pip install -r requirements.txt
